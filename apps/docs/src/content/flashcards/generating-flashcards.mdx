<br />
<Step title="Go to the Flashcards Page" number="2" />
<br />

Next, navigate to this documents flashcards page via the blue arrow on the flashcards section.

{/* step 3 */}

<br />
<Step title="Generate Flashcards" number="3" />
<br />

Next, select the "Generate" button in the top left corner of the flashcards page. You will then see a list of flashcards appear as each flashcard is generated.

<br />
