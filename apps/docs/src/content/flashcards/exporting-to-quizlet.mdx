_This guide assumes you have [generated flashcards](https://docs.studysail.com/generating-flashcards)._

In this guide we'll take the generated flashcards from a document on StudySail and import them as a new flashcard set on [Quizlet](https://quizlet.com).

{/* step 1 */}

<br />
<Step title="Copying Quizlet Compatible Format" number="1" />
<br />

First, on a given flashcard page, click the quizlet button among the options in the top right corner above the list of flashcards.

<br />
This will automatically copy a Quizlet compatible formatted text blob.

<ImageWithCaption
  src="https://firebasestorage.googleapis.com/v0/b/blog-files-d9a40.appspot.com/o/quizletbutton.gif?alt=media&token=d1047e53-5635-45f5-8eae-5f7f31d4acf6"
  alt="Quizlet button"
/>

{/* step 2 */}

<br />
<Step title="Create a New Study Set" number="2" />
<br />

Next [begin to create a new set](https://quizlet.com/create-set).

{/* step 3 */}

<br />
<Step title="Paste Flashcards" number="3" />
<br />

Next, click the import button:

<ImageWithCaption
  src="https://firebasestorage.googleapis.com/v0/b/blog-files-d9a40.appspot.com/o/quizletimport.png?alt=media&token=bfe4564d-b873-476d-be53-bb5502378b32"
  alt="quizlet import button"
/>

This should prompt you with a text box like this one:

<ImageWithCaption
  src="https://firebasestorage.googleapis.com/v0/b/blog-files-d9a40.appspot.com/o/quizlettextbox.png?alt=media&token=f2f77423-4a11-4a82-855a-2185f8dda21e"
  alt="quizlet import button"
/>

Leaving all of the settings in their default state, paste the format that you copied from StudySail:

<ImageWithCaption
  src="https://firebasestorage.googleapis.com/v0/b/blog-files-d9a40.appspot.com/o/quizletpaste.png?alt=media&token=d67a0ab3-ef32-4103-a096-14d383a0af8c"
  alt="quizlet import button"
/>

Finally, click the import button and then the create button to finalize your Quizlet set.

{/* step 4 */}

<br />
<Step title="Done!" number="4" />
<br />

You have successfully imported your StudySail flashcards into Quizlet. You can now utilize some of the awesome studying features that Quizlet has to offer!
